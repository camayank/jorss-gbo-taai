# Tax Expert Audit Contract
**AI Tax Advisor QA — 2–3 Week Engagement**

## What We Built

CA4CPA Global LLC has built an AI-powered tax advisor that:
- Guides taxpayers through 82 questions across income, deductions, retirement, real estate, and advanced strategies
- Calculates federal + state tax liability in real time
- Detects 30+ tax-saving opportunities using 764 rules (QBI, Augusta Rule, Defined Benefit, Cost Segregation, R&D Credits, etc.)
- Generates a full advisory report for CPA review

The platform targets $150K+ earners — self-employed, business owners, real estate investors, and high-W2 employees.

## What We Need

A **practicing Enrolled Agent (EA) or CPA** to run structured scenarios through the advisor and grade every response against what a competent tax professional would say.

This is not a technical role. You will use the advisor as a client would, then evaluate whether:
- The tax calculation is correct
- The right strategies appeared (and wrong ones didn't)
- IRS code references are accurate
- The conversation felt natural and trustworthy

## What to Grade

For each scenario, note:

**P0 (Must Fix — blocks CPA demo):**
- Wrong tax calculation (>5% error on federal tax)
- Missing strategy that clearly applies
- Incorrect IRS code citation

**P1 (Fix Before Launch):**
- Strategy appears but shouldn't (false positive)
- Explanation is confusing to a non-CPA
- Missing key follow-up question

**P2 (Nice to Have):**
- Wording could be sharper
- Ordering of strategies could be better
- Hint text incomplete
