## Variable Reference

All templates use placeholder variables in the format `{{VARIABLE_NAME}}`. Common variables include:

### Firm Variables
| Variable | Description |
|----------|-------------|
| `{{FIRM_NAME}}` | Your firm's name |
| `{{FIRM_PHONE}}` | Office phone number |
| `{{FIRM_EMAIL}}` | General contact email |
| `{{PORTAL_URL}}` | Client portal login URL |
| `{{SCHEDULING_LINK}}` | Calendly/scheduling link |
| `{{CPA_NAME}}` | Individual CPA's name |
| `{{CPA_PHONE}}` | Individual CPA's direct line |
| `{{CREDENTIALS}}` | CPA, EA, etc. |

### Client Variables
| Variable | Description |
|----------|-------------|
| `{{CLIENT_NAME}}` | Client's full name |
| `{{FIRST_NAME}}` | Client's first name |
| `{{CLIENT_ADDRESS}}` | Client's mailing address |
| `{{TAX_YEAR}}` | Relevant tax year |
| `{{STATE}}` | Client's state of residence |

### Financial Variables
| Variable | Description |
|----------|-------------|
| `{{TOTAL_SAVINGS}}` | Total identified tax savings |
| `{{CURRENT_TAX}}` | Current tax liability |
| `{{EFFECTIVE_RATE}}` | Effective tax rate % |
| `{{MARGINAL_RATE}}` | Marginal tax rate % |

### Service Variables
| Variable | Description |
|----------|-------------|
| `{{SIMPLE_PRICE}}` | Simple return pricing |
| `{{STANDARD_PRICE}}` | Standard return pricing |
| `{{MODERATE_PRICE}}` | Moderate return pricing |
| `{{COMPLEX_PRICE}}` | Complex return pricing |

## Best Practices

### Personalization
- Always use the client's first name
- Reference specific details from their situation
- Include concrete numbers (savings identified, tax rates, etc.)

### Timing
- Send pre-season reminders in early January
- Follow up on missing documents within 1 week
- Deliver drafts within promised timeframe
- Send year-end reminders by mid-November

### Branding
- Use consistent colors, fonts, and logo placement
- Include your scheduling link in every communication
- Add your phone number for urgent matters

### Compliance
- Include appropriate disclaimers on advisory materials
- Keep engagement letters updated with current terms
- Document all client communications
